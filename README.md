### 🎭 [AI-Jubensha](https://github.com/Zewang0217/ai-jubensha) — 多 Agent 沉浸式剧本杀引擎

```
让 AI 扮演角色，让玩家分不清真假。
```

[![Stars](https://img.shields.io/github/stars/Zewang0217/ai-jubensha?style=flat-square&color=5E81AC&label=⭐)](https://github.com/Zewang0217/ai-jubensha)
[![Lang](https://img.shields.io/badge/Java-Spring%20Boot-brightgreen?style=flat-square)](https://github.com/Zewang0217/ai-jubensha)

**核心架构：**
- 🧠 **四级 Agent 协作**：DM（主持人）→ Player（玩家）→ Judge（裁判）→ Summary（复盘），每个 Agent 有独立 System Prompt 和记忆上下文
- 💾 **混合记忆引擎**：Caffeine L1 缓存 + Milvus 向量数据库 RAG，解决 LLM "金鱼记忆"，支持跨会话长期记忆
- ⚡ **流式剧本生成**：Spring Boot 3 + Reactive Streams，剧本边写边播，WebSocket 实时推送到前端
- 🎨 **沉浸式前端**：React + 剧本 UI 组件库，自动渲染角色对话、线索卡、投票面板

**技术栈：** `Java 21` · `Spring Boot 3` · `LangChain4j` · `Milvus` · `React` · `WebSocket`

<br/>

### 🐕 [Cyber-Stray](https://github.com/Zewang0217/cyber-stray) — 赛博街溜子 · 自动化信息猎犬

```
一只在互联网上游荡的 Agent，自主觅食、消化、沉淀。
```

[![Stars](https://img.shields.io/github/stars/Zewang0217/cyber-stray?style=flat-square&color=5E81AC&label=⭐)](https://github.com/Zewang0217/cyber-stray)
[![Lang](https://img.shields.io/badge/TypeScript-Node.js-blue?style=flat-square)](https://github.com/Zewang0217/cyber-stray)

**设计哲学：** 不是爬虫，是"猎犬"—— 有目标、有判断、有记忆、有反思。

**能力矩阵：**
- 🎯 **自主目标设定**：基于用户兴趣画像，Agent 自主决定"今天去哪逛、逛什么"
- 🔍 **多源信息采集**：浏览器自动化 + API 调用 + RSS 聚合，覆盖技术博客、GitHub Trending、论文站点
- 🧠 **智能筛选过滤**：本地 LLM 评估信息价值，自动丢弃低质量内容
- 📚 **知识沉淀**：有价值的信息自动分类、摘要、入库，构建个人知识图谱

**技术栈：** `TypeScript` · `Playwright` · `LangChain` · `SQLite` · `Docker`

### ⚔️ [Agents-Debate](https://github.com/Zewang0217/agents-debate) — 辩论式 PRD 生成器

```
两个 Agent 吵架吵出来的需求文档，比你一个人写的更全面。
```

两个 LLM Agent 分别扮演"产品经理"和"技术负责人"，围绕同一个产品概念展开多轮辩论。通过对抗性思维，自动暴露逻辑漏洞、边界情况和潜在风险，最终生成一份经过"压力测试"的 PRD。

**技术栈：** `Python` · `LangChain` · `Multi-Agent Debate`

<br/>

### 🤖 ToThink — GUI Agent 自动化平台 *(私有仓库)*

基于视觉感知的层次化多智能体框架，让 AI 像人类一样操作电脑。

- **层次化架构**：Planner（规划）→ Perceiver（感知）→ Actor（执行）三级 Agent 协作
- **视觉感知**：OCR + 元素检测，精准定位按钮、输入框、菜单
- **场景覆盖**：Chrome 自动化、Word 文档处理、微信消息发送等
- **Electron 桌面端**：可视化任务编排，支持多设备同步

**技术栈：** `TypeScript` · `Electron` · `Playwright` · `OpenCV` · `OCR`
